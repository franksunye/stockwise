# 01. 产品故事

> **从"赌徒"到"狙击手"** —— 那个让用户彻底爱上产品的瞬间

---

## 🎯 Aha Moment

**"算清楚帮你少亏了多少钱"的瞬间，就是产品的 100 亿美元时刻。**

---

## 📖 三幕剧

### 第一幕：混乱

```
下午 2:30，港股跳水，从 15.2 跌到 14.8
用户阿强，心跳加速，满手是汗
```

**脑中两个声音：**
- 🟢 贪婪：*"是假摔，主力洗盘，拿住！"*
- 🔴 恐惧：*"万一崩了？要不要跑？"*

**他在赌运气。**

---

### 第二幕：干预

阿强点开 App，屏幕极简冷酷：

```
┌─────────────────────────────┐
│  状态：🔴 触发止损           │
│  现价：14.76                │
│  你的规则：跌破 14.78 离场   │
│  今日建议：坚决卖出          │
└─────────────────────────────┘
```

没有 K 线，没有新闻，没有情绪。

他看着 **"你的规则"** 四个字 —— 昨晚冷静时设的红线。

**他卖了。**

---

### 第三幕：顿悟 💡

第二天，股票继续暴跌至 **13.50**。

复盘页弹出：

```
┌─────────────────────────────────────────┐
│  ✅ 恭喜。你没有预测对行情，             │
│     但你遵守了纪律。                     │
│                                         │
│  本次执行，避开了 8.5% 的额外亏损。      │
└─────────────────────────────────────────┘
```

**Aha！**

*"这个 App 不是用来帮我赚大钱的，它是用来救我命的。"*

---

## 💎 产品精髓

| 竞品 | StockWise |
|------|-----------|
| 承诺 **"我帮你赢"** | 承诺 **"我是你的理性锚点"** |
| 制造焦虑 | 消除噪音 |
| 预测涨跌 | 执行纪律 |

---

## 🔑 核心价值主张

> **不做预测涨跌的水晶球，做执行纪律的抬头显示器。**
