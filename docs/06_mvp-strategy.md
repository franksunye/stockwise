
## 📊 数据策略

### 核心股票池 (50 只)

**选择标准**:
- 港股通标的
- 日均成交额 > 5000 万
- 市值 > 100 亿港币

**示例**:
```
02171  映客
02269  药明生物
01801  信达生物
00700  腾讯控股
09988  阿里巴巴
...
```

### 更新策略

- **频率**: 每日 18:00 (收盘后)
- **范围**: 仅更新 50 只核心股票
- **成本**: Turso 免费额度内
