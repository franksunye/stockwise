# 02. 产品规格

> **MVP 功能模块** —— 如果一个功能让用户思考超过 3 秒，砍掉它。

---

## 🎯 产品形态

- **类型**：Mobile-First PWA (可添加到手机桌面)
- **视觉**：深色金融风格，高对比度红/绿/黄信号

---

## 📱 功能模块

### A. 参数锚点 (Settings)

> 规则的制定 —— 用户的"冷静期"操作

**存储**：LocalStorage (无需登录，隐私极快)

| 输入项 | 必填 | 示例 | 说明 |
|--------|------|------|------|
| 止损/支撑位 | ✅ | 14.78 | 跌破无条件离场 |
| 压力/突破位 | ❌ | 15.80 | 突破且放量才试错 |
| 最低量能要求 | ❌ | 500万 | 过滤缩量信号 |

---

### B. 决策看板 (Dashboard)

> 执行的时刻 —— 单屏卡片流

**布局**

```
┌─────────────────────────────────┐
│  ☰    [股票选择器]         ⚙️   │
├─────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐       │
│  │  现价   │  │ 关键位  │       │
│  │  涨跌幅 │  │  状态   │       │
│  └─────────┘  └─────────┘       │
│  ┌─────────┐  ┌─────────┐       │
│  │  成交量 │  │   RSI   │       │
│  └─────────┘  └─────────┘       │
├─────────────────────────────────┤
│       [ AI 决策结论 ]           │
│    "今日建议：观望"             │
├─────────────────────────────────┤
│                           📷    │
└─────────────────────────────────┘
```

**信号算法**

| 信号 | 条件 | 样式 |
|------|------|------|
| 🟢 **买入/持有** | 现价 > 支撑位 (且 > 压力位，若有) | 绿底 |
| 🔴 **卖出/止损** | 现价 < 支撑位 | 红底 + 呼吸动画 |
| 🟡 **观望** | 价格夹在支撑与压力之间 | 黄边 |

---

### C. 智能复盘 (Review)

> 顿悟的瞬间 —— 每日对比"昨日决策"与"今日结果"

**统计**

| 指标 | 示例 |
|------|------|
| 策略胜率 | 40% |
| 累计避险/收益 | +12.5% ✨ |

**时间轴**

| 场景 | 展示 |
|------|------|
| 建议卖出 → 次日跌 | ✅ **避险 5%** (绿) |
| 建议买入 → 次日跌 | ❌ **浮亏 2%** (红) |
| 建议观望 → 次日涨 | ⚪ **踏空 3%** (灰) |

---

### D. 视觉分析 (Photo Analysis)

> AI 辅助 —— K 线截图智能分析

**交互**

1. 点击 📷 FAB 按钮
2. 上传 K 线截图
3. AI 返回趋势判断 + 操作建议

**AI 人设**

> *"你是严格的风控官。如果跌破关键位，只回一个字：跑。"*

---

## 🚀 开发路线图

| 优先级 | 里程碑 | 说明 |
|--------|--------|------|
| **P0** | 数据管道 | Python 脚本 + Turso 写入 |
| **P1** | HUD 骨架 | 显示现价 + 红绿灯 |
| **P2** | 规则引擎 | LocalStorage 动态规则 |
| **P3** | 复盘闭环 | 计算避险收益 |
| **P4** | PWA + AI | Manifest 配置 + 图片分析 |

---

## ⚠️ 边界情况

| 场景 | 处理 |
|------|------|
| 无数据/休市 | 显示最近交易日数据 |
| 未设置规则 | 引导卡片 "点击设置风控锚点" |
| 网络错误 | Toast 提示 + 下拉刷新 |
